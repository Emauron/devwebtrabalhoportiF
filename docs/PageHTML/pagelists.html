Listas

Lista ordenada - <ol>

As listas ordenadas são usadas quando você deseja apresentar itens em uma sequência específica, geralmente numerando-os ou usando letras. Os itens em uma lista ordenada são automaticamente numerados ou marcados, dependendo do tipo de lista (números, letras, algarismos romanos, etc.). A tag principal para criar listas ordenadas é <ol>, e cada item da lista é representado por uma tag <li> (List Item).

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>

Lista não ordenada - <ul>

As listas não ordenadas são usadas para listar itens em uma ordem não específica, geralmente usando marcadores como pontos, traços ou círculos.
A tag principal para criar listas não ordenadas é <ul>, e cada item da lista também é representado por uma tag <li>.

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Lista de definição - <dl>

As listas não ordenadas são usadas para listar itens em uma ordem não específica, geralmente usando marcadores como pontos, traços ou círculos.
A tag principal para criar listas não ordenadas é <ul>, e cada item da lista também é representado por uma tag <li>.

<dl>
<dt>HTML</dt>
<dd>HyperText Markup Language - Linguagem de Marcação de Hipertexto.</dd>
<dt>CSS</dt>
<dd>Cascading Style Sheets - Folhas de Estilo em Cascata.</dd>
</dl>
