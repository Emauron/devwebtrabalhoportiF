ID E CLASS

id

Identificador Único: Um ID (Identificação) é um atributo que você pode adicionar a um elemento HTML para torná-lo exclusivo na página. Cada ID deve ser único em todo o documento. Isso significa que nenhum outro elemento na mesma página pode ter o mesmo ID.
Os IDs são usados principalmente quando você deseja selecionar um elemento HTML específico para aplicar estilos a ele ou manipulá-lo com JavaScript. Você pode selecionar elementos por ID usando o seletor #, seguido do nome do ID.

<div id="meu-elemento-unico">
Este é um elemento com um ID.
</div>

class

Identificador Não-Único: Uma classe é um atributo que você pode adicionar a um ou mais elementos HTML para agrupá-los ou identificá-los de acordo com algum critério. Ao contrário dos IDs, você pode usar a mesma classe em múltiplos elementos na mesma página.
Classes são frequentemente usadas quando você deseja aplicar estilos a um grupo de elementos semelhantes. Elas permitem que você estilize vários elementos com a mesma classe usando um único seletor

<p class="classe1">
Pertence a classe1.
</p>
<p class="classe1">
Também pertence a classe 1.
</p>
