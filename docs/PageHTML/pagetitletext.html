TÍTULOS E TEXTO

Títulos

Em HTML, os títulos são elementos usados para definir a estrutura hierárquica de uma página da web e fornecer informações sobre a importância relativa do conteúdo. Os títulos são essenciais para a acessibilidade, usabilidade e otimização de mecanismos de busca (SEO) de uma página. Existem seis níveis de títulos em HTML, representados pelas tags <h1>, <h2>, <h3>, >h4<, <h5>, e <h6>. Aqui está uma descrição de cada um: <h1> (Heading 1): O título principal da página. Deve ser usado uma única vez em cada página e representa o tópico mais importante da página. <h2> (Heading 2): Usado para subtítulos importantes que estão relacionados ao título principal (<h1>). Pode haver vários <h2> em uma página, mas eles devem se subordinar ao <h1>. <h3> (Heading 3): Usado para subdividir tópicos dentro de um <h2>. Ou seja, <h3>s são subordinados a <h2>s e devem ser usados para seções mais específicas. <h4>, <h5>, e <h6>: São usados para títulos de níveis mais baixos, cada um sendo menos importante que o nível acima. A estrutura de títulos deve formar uma hierarquia lógica.

Árvore de Elementos: O DOM representa uma página da web como uma árvore de elementos, onde cada elemento HTML (como tags <html>, <head>, <body>, <div>, etc.) é representado por um nó na árvore. Esses nós estão organizados hierarquicamente, refletindo a estrutura do documento.
