Âncora

A tag âncora em HTML é representada pela tag <a>, que é usada para criar hiperlinks ou links em uma página da web. Os hiperlinks permitem que os usuários cliquem em um texto ou imagem para navegar para outra página, recurso ou recurso na web, seja em outro site, na mesma página ou em um local diferente na mesma página. Aqui estão os principais atributos e usos da tag <a>

Atributo href (Hypertext Reference): Especifica o destino do link, ou seja, a URL para a qual o usuário será direcionado quando o link for clicado. A URL pode ser relativa (para links internos) ou absoluta (para links externos).

Atributo target: Este atributo controla como o destino do link será exibido. Os valores comuns incluem:

  • _blank: Abre o link em uma nova janela ou guia do navegador.
  • _self: Abre o link na mesma janela ou guia (padrão).
  • _parent: Abre o link na janela pai.
  • _target: Remove todos os frames e abre o link na janela superior.


Texto do Link: O texto ou conteúdo dentro das tags <a> é o que os usuários veem e clicam para seguir o link. É conhecido como o "texto âncora".

<a href="https://www.exemplo.com" target="_blank">
