DOM

O DOM, ou Document Object Model, em português "Modelo de Objeto de Documento", é uma representação estruturada e hierárquica de um documento HTML (ou XML) que permite que os programas e scripts acessem e interajam com os elementos da página web. Em outras palavras, o DOM é uma interface que permite que os desenvolvedores manipulem o conteúdo, a estrutura e o estilo de uma página da web usando HTML, CSS, código JavaScript ou outras linguagens.

Árvore de Elementos: O DOM representa uma página da web como uma árvore de elementos, onde cada elemento HTML (como tags <html>, <head>, <body>, <div>, etc.) é representado por um nó na árvore. Esses nós estão organizados hierarquicamente, refletindo a estrutura do documento.
