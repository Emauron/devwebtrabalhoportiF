Imagens

Em HTML, as imagens são incorporadas em uma página da web usando a tag <img>. A tag <img> não tem uma tag de fechamento, pois é uma tag vazia que não contém conteúdo textual. Em vez disso, você define os atributos dentro da tag <img> para especificar a localização e o comportamento da imagem.
ATRIBUTOS:


  • src: Especifica o caminho para a imagem que você deseja exibir. Pode ser uma URL da web ou um caminho local para a imagem no seu servidor.
  • alt: O atributo alt (texto alternativo) fornece uma descrição textual da imagem. É importante para acessibilidade, pois é exibido quando a imagem não pode ser carregada e é lido por leitores de tela para ajudar os usuários com deficiência visual a entender o conteúdo da imagem.
  • title: Exibirá um texto de dica quando o usuário passar o mouse sobre a imagem.
<img src="caminho/para/sua/imagem.jpg" alt="Descrição da imagem" title="Imagem de exemplo">
