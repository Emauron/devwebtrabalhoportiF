BACKGROUND

background-color

O background-color é uma propriedade CSS que define a cor de fundo de um elemento HTML. A propriedade background-color aceita valores que especificam a cor desejada. Você pode usar nomes de cores pré-definidos, códigos de cores em hexadecimal, códigos RGB ou RGBA, entre outros formatos de cores suportados pelo CSS.

<head>
<title>Minha Página</title>
<link rel="stylesheet" type="text/css" href="styles.css">
</head>

background-image

A propriedade background-image em CSS permite que você defina uma imagem como plano de fundo para um elemento HTML. Ela é amplamente utilizada para adicionar elementos visuais, texturas ou imagens de fundo a elementos como divs, seções, cabeçalhos, ou mesmo o corpo inteiro de uma página da web.
A propriedade background-image aceita um valor que especifica o caminho (URL) da imagem que você deseja usar como plano de fundo.

elemento {
background-image: url('caminho/para/sua-imagem.jpg');
background-repeat: no-repeat; /* Não se repete */
background-size: cover; /* Cobrir todo o elemento */ background-position: center; /* Centralizar a imagem no elemento */
background-attachment: fixed; /* Fixa a imagem de fundo */
}
