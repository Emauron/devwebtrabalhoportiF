BORDA

A propriedade border em CSS é usada para definir a borda de um elemento HTML. Ela permite que você adicione uma linha ou um contorno ao redor do elemento, ajudando a destacá-lo, separá-lo de outros elementos e melhorar o design geral de uma página da web.

Seletor: Isso especifica qual elemento ou elementos HTML a regra se aplicará. Você pode usar tags HTML, classes, IDs ou outros seletores para direcionar elementos específicos. Por exemplo, p para parágrafos, .classe para classes, #id para IDs.

Propriedade: É a característica que você deseja estilizar, como color (cor), font-size (tamanho da fonte), background-color (cor de fundo), etc.

Valor: É o valor que você deseja atribuir à propriedade. Pode ser um número, uma cor, um tamanho, uma unidade de medida, etc.

elemento {
border-style: solid;/* Borda sólida */
border-style: dashed;/* Borda tracejada*/
border-color: white /* Cor da borda */
border-top: / border-bottom: / border-right: / border-left: /* Posição da borda */
}
