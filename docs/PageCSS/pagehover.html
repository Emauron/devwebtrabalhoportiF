:hover

O :hover é um pseudo-classe em CSS que permite que você aplique estilos a um elemento HTML quando o cursor do mouse é colocado sobre ele. Essa pseudo-classe é amplamente usada para criar interatividade e realçar elementos em páginas da web quando os usuários interagem com eles.
A sintaxe básica para usar :hover é adicionar :hover ao seletor de um elemento no seu arquivo CSS.

elemento:hover {
*alterações nas propriedades do elemento*
}

É muito usado para mudar a cor ou a cor do fundo de elementos, realçar elemtos, mostrar informações adicionais, criar animações simples, alterar o layout, esconder ou mostrar elemento e por ai vai.
