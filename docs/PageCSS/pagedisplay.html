DISPLAY

O atributo display é usado para controlar como um elemento HTML é exibido no layout da página. Ele determina o tipo de "caixa" que um elemento se torna e como ele interage com outros elementos em relação ao espaço na página.

none

Elementos com display: none; são completamente ocultados da renderização da página. Eles não ocupam espaço na página e não são interativos.

elemento {
display: none;
}

block

Elementos com display: block; são exibidos como blocos retangulares que ocupam toda a largura disponível. Eles começam em uma nova linha e empurram elementos subsequentes para uma nova linha também.

elemento {
display: block;
}

inline

Elementos com display: inline; são exibidos em linha com o texto ou outros elementos em torno deles. Eles não começam em uma nova linha e só ocupam o espaço necessário.

elemento {
display: inline;
}

inline-block

Elementos com display: inline-block; são exibidos em linha, mas como blocos retangulares, permitindo que você ajuste largura e altura. Eles não começam em uma nova linha e podem ocupar espaço horizontalmente, como elementos inline, mas também podem ter dimensões verticais como elementos block.

elemento {
display: inline-block;
}

flex

O display: flex é uma propriedade do CSS que define um elemento pai (contêiner) como um container flexível, permitindo o posicionamento e o alinhamento de seus filhos (itens) de forma flexível em uma única dimensão, que pode ser horizontal (linha) ou vertical (coluna).

flex-direction: row; row-rever; column; column-reverse | direção dos elementos

justify-content: flex-start; flex-end; center; space-around; space-beteween; stretch; | alinhamento dos elementos

align-items: flex-start; flex-end; center; stretch | alinhamentos dos elementos

elemento {
display: flex;
flex-direction: ;
justify-content: ;
align-items: ;

}

grid layout

é uma poderosa técnica de layout em CSS que permite criar layouts bidimensionais (ou seja, em duas dimensões: colunas e linhas) de maneira altamente flexível e precisa. O Grid é particularmente útil para criar layouts complexos e responsivos, onde os elementos precisam ser colocados em relação uns aos outros de forma precisa e eficiente.

flex-direction: row; row-rever; column; column-reverse | direção dos elementos

justify-content: flex-start; flex-end; center; space-around; space-beteween; stretch; | alinhamento dos elementos

align-items: flex-start; flex-end; center; stretch | alinhamentos dos elementos

Possui diversas propriedades, como:

elemento {
display: grid;
grid-template-columns: ;
grid-template-rows: ;
gap: ;

}
