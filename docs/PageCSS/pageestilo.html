ESTILO

Primeiro, crie um arquivo CSS separado. Você pode fazer isso criando um novo arquivo de texto com a extensão ".css". Por exemplo, "styles.css". Este arquivo conterá todas as regras de estilo para sua página da web.Para que o navegador aplique os estilos definidos em seu arquivo CSS, você precisa vinculá-lo à sua página HTML. Isso é feito usando a tag <link> no elemento <head> do seu documento HTML.

Seletor: Isso especifica qual elemento ou elementos HTML a regra se aplicará. Você pode usar tags HTML, classes, IDs ou outros seletores para direcionar elementos específicos. Por exemplo, p para parágrafos, .classe para classes, #id para IDs.

Propriedade: É a característica que você deseja estilizar, como color (cor), font-size (tamanho da fonte), background-color (cor de fundo), etc.

Valor: É o valor que você deseja atribuir à propriedade. Pode ser um número, uma cor, um tamanho, uma unidade de medida, etc.

<head>
<title>Minha Página</title>
<link rel="stylesheet" type="text/css" href="styles.css">
</head>
.seletor {
propriedade: valor;
}

#container {
background-color: blue;
}
